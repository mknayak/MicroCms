# Architecture Decision Records

This directory contains all Architecture Decision Records (ADRs) for MicroCMS.

ADRs document **significant decisions** made during development — what was decided, why, and what was rejected.  
They are immutable once accepted. Superseded records are marked but never deleted.

## Format

Each ADR follows the [MADR 3.0](https://adr.github.io/madr/) template:
`Status → Context → Decision → Consequences → Alternatives considered`
